# Vanilla DINO

**Version:** 1.0.0 | **Type:** content | **Author:** DINOForge Team

Canonical reference pack containing all 100+ vanilla DINO units, 6 factions, buildings, weapons, and doctrines. Use this as the base for all mod unit/...
