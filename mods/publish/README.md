# Published mods (local store)

`.civmod` archives copied here by `POST /control/mods/publish` for local distribution. Files are named `<manifest.meta.id>.civmod` and are gitignored; only this README is tracked.

Install from the dashboard or `POST /control/mods/install` with `source` set to `mods/publish/<id>.civmod`. List entries via `GET /control/mods/published`.
