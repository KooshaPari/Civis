# Civ-Sim Glossary and Symbol Table

## Glossary
1. `run`: one deterministic simulation execution over ticks.
2. `tick`: atomic time-step for state transition.
3. `policy bundle`: versioned intervention set evaluated by policy engine.
4. `lod`: level-of-detail view over the same underlying state.
5. `legitimacy`: governance trust metric used in institutional transitions.
6. `capture`: degree of institutional control by narrow interests.
7. `subsistence floor`: minimum guaranteed transfer/provisioning level.
8. `shadow network`: hidden influence/flow layer affecting governance outcomes.

## Symbols
1. `S_t`: simulation state at tick `t`.
2. `u_t`: control vector emitted by policy evaluator at tick `t`.
3. `M_t`: metrics snapshot at tick `t`.
4. `E_t`: net energy stock/availability at tick `t`.
5. `L_t`: legitimacy metric at tick `t`.
6. `C_t`: capture metric at tick `t`.
7. `D_t`: climate damage estimate at tick `t`.
8. `R_t`: resilience/adaptation capacity at tick `t`.
