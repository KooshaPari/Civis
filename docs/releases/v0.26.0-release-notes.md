## Overview

DINOForge v0.26.0 is a **comprehensive platform maturity release** focusing on user experience, developer tooling, and production-grade observability. This release ships **35+ commits** across 10 major feature areas, with **zero breaking changes** and full backward compatibility.

## Major Features

### 1. CLI & Workflow Automation

#### Single-Command Development Loop
```bash
dinoforge build         # Compile pack + DLL
dinoforge deploy        # Copy to game BepInEx
dinoforge relaunch      # Kill + relaunch game
dinoforge smoke         # Run quick validation (all 4 in sequence)
```

Patterns match **WorldSphereMod wsm3d** conventions for familiar DX across mod platforms.

#### Enhanced Pack Tools
- **`dinoforge pack diff <a> <b>`** — Side-by-side visual comparison of pack manifests
- **`dinoforge pack thunderstore <pack>`** — Package for Thunderstore marketplace submission
- **`dinoforge dev-tools install`** — Download + register UnityExplorer for runtime inspection

#### Build Robustness
- **`--clean` flag** — Solves Pattern #233 stale obj/ cache bugs via aggressive cleanup
- **System.CommandLine 2.x compatibility** — Future-proofs CLI against SDK changes

## Migration Guide

### From v0.25.0 to v0.26.0

**Good news**: No breaking changes. All packs, configs, and saves are 100% compatible.

#### Recommended Actions

1. **Update BepInEx Config** (Optional)
   ```ini
   # New settings available in BepInEx.cfg:
   ShowDebugOverlayOnStart = true        # F9 on game start
   EnableHotReload = true                # Pack hot-reload watching
   HotReloadDebounceMs = 500             # Reload delay (ms)
   LogLevel = Info                       # Verbosity
   ```

2. **Try New CLI Commands**
   ```bash
   # Single workflow command replaces 4 manual steps
   dinoforge smoke
   ```

3. **Explore F10 Enhancements**
   - Press F10 to open mod browser
   - Try search, filter, sort
   - Click "Settings" in detail pane for per-pack tuning
   - View "Telemetry" tab for performance insights

4. **Sign Packs for Distribution** (Optional)
   ```bash
   dinoforge pack sign <pack-id> --key private.pem
   dinoforge pack verify <pack-id> --key public.pem
   ```

## Upgrade Instructions

### For Users

1. **Automatic** — Steam auto-updates DINO game
2. **DINOForge Runtime** — Next game launch auto-patches from BepInEx loader
3. **CLI** — `dotnet tool update -g dinoforge` (if installed globally)

### For Modders

1. **Update CLI**
   ```bash
   dotnet tool update -g dinoforge
   ```

2. **Update SDK NuGet** (if consuming SDK in your mod DLL)
   ```bash
   dotnet add package DINOForge.SDK --version 0.26.0
   ```

3. **No code changes needed** — v0.25 packs load unchanged in v0.26

### For Contributors

1. **Clone/sync** latest main branch
2. **Build**: `dotnet build src/DINOForge.sln -c Release`
3. **Deploy**: `dotnet build src/Runtime/DINOForge.Runtime.csproj -c Release -p:DeployToGame=true`
4. **Test**: Launch game, press F10 to verify mod browser
