## Documentation Guidelines

Each specification must include:
- **Summary**: One-sentence overview of scope
- **CIV Sim Integration Notes**: How this spec integrates with the core simulation
- **API**: Clear interfaces and data structures
- **Examples**: Working examples where applicable
- **Validation**: How to verify correct implementation
