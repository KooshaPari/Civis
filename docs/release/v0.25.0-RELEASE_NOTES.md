## Upgrade Notes

### DINOForge.Runtime Multi-Target

Runtime DLL is now multi-targeted:
```xml
<TargetFrameworks>netstandard2.0;net8.0</TargetFrameworks>
```

- **netstandard2.0**: BepInEx plugin load (Mono 4.0 CLR)
- **net8.0**: Test/tool consumers with modern CLR

**Consumer impact**: None. NuGet resolution automatically picks correct TFM.

### Log Rotation

`WriteDebug()` output now rotates at 100 MB:
- Current log ≥ 100 MB → renamed to `.1` (overwrites prior)
- Fresh log created
- Append failures silently fallback to BepInEx logger

**Manual cleanup**: Remove stale `.1` files in `BepInEx/` before next game launch.
